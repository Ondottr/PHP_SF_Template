/* TECHNOLOGIES */
This website was built using the following technologies:
— PHP Simple Framework with Symfony and Doctrine ORM
— Codeception and PHPUnit for testing
— HTML5 and CSS3
— JavaScript, jQuery, and WebPack
— PostgreSQL and MySQL/MariaDB databases
— FASTPANEL server management panel
— Debian 11 to 13 used on all servers and as a desktop OS for work
— Servers hosted on Contabo

For more information about these technologies, please visit:
- [PHP Simple Framework](https://github.com/Ondottr/PHP_SF_Template)
- [Symfony](https://symfony.com/)
- [Doctrine ORM](https://www.doctrine-project.org/)
- [FASTPANEL](https://fastpanel.direct/)
